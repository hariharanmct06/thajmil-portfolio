# Premium Futuristic Portfolio - Thajmil Akthar

A cinematic, futuristic, dark-themed personal portfolio website designed for **Thajmil Akthar** (Aerospace Engineering Student, AI Developer, and Automation Enthusiast).

## 🚀 Live Demonstration
The website runs as a fully static, client-side application. 
To run locally:
```bash
# Using python HTTP server
python -m http.server 8000
```
Then navigate to `http://localhost:8000`.

## 🎨 Design Features
*   **Space & Tech Theme**: Dark space background (`#040711`) highlighted with glowing gold accents (`#fbc02d`) and technical telemetry widgets.
*   **Glassmorphism**: Elegant translucent navigation panels and grid cards with background blurs.
*   **3D Interactive Tilt**: Competencies cards track the cursor to tilt in a 3D perspective plane and project a dynamic light spot.
*   **Animated Timeline**: Chronological journey nodes that slide and pulse into view upon scrolling.
*   **Achievements Panel**: Counter statistics that automatically increment when visible.
*   **Telemetry Details Modal**: Clickable project showcase cards showing full system descriptions and stack tags.

## 📂 Codebase Structure
*   `index.html` - Semantic layout grid and components.
*   `style.css` - Custom styling rules, variables, animations, and responsiveness.
*   `script.js` - Logic for cursor glow, card tilts, timeline tracking, and modal popups.
*   `assets/thajmil.jpeg` - Portrait image.
